## Naming Convention

**Format**: `NLP[XX]_[Y]_[Description].ipynb`

Where:
- XX = Module number (01-14)
- Y = Optional sub-notebook number
- Description = Descriptive name with underscores or hyphens

Examples:
- NLP09_Transfer_Learning_Demo.ipynb
- NLP11_1_transformer-anatomy.ipynb
- NLP14_2_Modern_LLMs.ipynb
